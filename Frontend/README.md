# MyAccess Frontend

Aplicación React Native con Expo para gestión de acceso educativo.

## Instalación

```bash
cd Frontend
npm install
```

## Ejecutar

```bash
npm start
```

## Estructura

```
src/
├── components/       # Componentes reutilizables
├── screens/         # Pantallas de la app
├── navigation/      # Configuración de navegación
└── theme/          # Design tokens (colores)
```

## Flujo de Usuario

1. Onboarding → Pantalla de bienvenida
2. Login → Inicio de sesión
3. Register → Crear cuenta
4. Verification → Código OTP
